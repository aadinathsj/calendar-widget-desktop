# Calendar Widget Setup Guide

## Simple 3-Step Setup (No Technical Knowledge Required!)

### Step 1: Extract the Files
1. Find the `calendar-widget.zip` file you downloaded
2. **Right-click** on the zip file
3. Select **"Extract All..."**
4. Click **"Extract"**
5. A new folder will open with all the files

### Step 2: Create Desktop Shortcut
1. In the extracted folder, find the file named **"calendar-widget.exe"**
2. **Right-click** on `calendar-widget.exe`
3. Select **"Send to"** → **"Desktop (create shortcut)"**
4. A shortcut will appear on your desktop

### Step 3: Run the Widget
1. **Double-click** the shortcut on your desktop
2. The widget will open in the corner of your screen
3. That's it! You're done! 🎉

## First-Time Use

When you open the widget for the first time:
- It will check if **Microsoft Outlook** is installed
- Make sure Outlook is **running** (it can be minimized)
- The widget will load your calendar automatically

## Daily Use

**To Open:** Double-click the desktop shortcut  
**To Close:** Click the ✕ button on the widget  
**To Minimize:** Click the − button on the widget

## Common Questions

**Q: The widget says "Outlook not found"**  
A: Make sure Microsoft Outlook is installed and running on your computer.

**Q: I don't see any meetings**  
A: Click the refresh (↻) button in the top-right corner.

**Q: How do I make it start automatically when I turn on my computer?**  
A:
1. Press **Windows Key + R**
2. Type: `shell:startup` and press Enter
3. Copy the desktop shortcut into this folder
4. The widget will now start automatically!

**Q: Where are my meeting notes saved?**  
A: Notes are saved in: `C:\Users\YOUR-USERNAME\AppData\Roaming\calendar-widget\meeting-notes\`

## Need Help?

If something isn't working:
1. Make sure Microsoft Outlook is running
2. Try closing and reopening the widget
3. Click the refresh button

# Calendar Widget - Complete Feature Guide

## ✨ What's New

### Glassmorphic Design
Beautiful frosted glass effect with semi-transparent backgrounds and backdrop blur - inspired by modern UI design principles.

**Visual Features:**
- Transparent window with CSS `backdrop-filter: blur(20px)`
- Semi-transparent cards (70% opacity white)
- Subtle borders and shadows
- Purple accent theme (like meditation apps)
- Smooth animations (200ms transitions)
- Modern, clean interface matching Windows 11

### Tab Navigation
Switch between two main views:
- **📅 Calendar**: View your Outlook meetings
- **⚡ Actions**: Manage quick-access bookmarks

### Actions Feature (NEW!)
Create bookmarkable actions - quick links to frequently used URLs.
